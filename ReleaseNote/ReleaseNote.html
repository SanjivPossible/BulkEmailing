beeEmailing: Bulk Emailing

November 2021 (version 1)

Welcome to the November 2021 release of bEmailing. There are many updates in this version that we hope you will like, some of the key highlights include:
  • SMTP configuration - Application will allow you to configurate any SMTP (Gmail, Yahoo, any organization SMTP detail).
  • Select data from Excel sheet - Application will allow to select excel sheet with required columns to add in To, Cc, Bcc, body,subject.
  • Send bulk email on single click - Application will instantly start sending email to recipients.

November 2021 (version 1.1)

Update 1.1.0: The update addresses these issues.
  • Image support - An image link with absolute file path will automatically converted to text while sending email in html format.
  • Reset Attachment - Remove selected attachment.
  • Progress Bar - Percentage of email send.
